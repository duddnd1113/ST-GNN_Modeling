# Appendix C3: Feature Scenario 정의

본 연구에서는 ST-GNN 입력 특성 조합을 체계적으로 검토하기 위해  
10가지 Feature Scenario를 설계하였다.  
모든 시나리오는 **기본 기상 변수 4종 + PM10**을 공통으로 포함하며,  
추가 특성 그룹의 포함 여부로 구분된다.

---

## 특성 그룹 정의

| 그룹 | 변수 | 차원 |
|------|------|:----:|
| **기본 기상** *(공통)* | 풍향_10m, 풍속_10m, 동서 방향 풍속, 남북 방향 풍속 | 4 |
| **PM10** *(타깃, 공통)* | PM10 | 1 |
| **오염물질** | SO2, CO, O3, NO2 | 4 |
| **기상변수** | 기온, 상대습도, 이슬점, 현지 기압, 해면 기압 | 5 |
| **강수변수** | 강수 유무, 60분 강수량, 일 누적 강수량 | 3 |
| **가시거리** | 가시거리 | 1 |
| **Mask — PM10** | PM10_interp_mask, PM10_spatial_mask | 2 |
| **Mask — 오염물질 전체** | SO2/CO/O3/NO2 × (interp + spatial) mask | 8 |
| **Mask — 요약** | any_interp_mask, any_spatial_mask | 2 |

> **Mask 변수**: 해당 시점 값이 결측 보간(interp) 또는 공간 보간(spatial)으로  
> 채워진 경우 1, 실측인 경우 0. 학습 손실 다운웨이팅에 활용.

## 시나리오 설계 의도

| 비교 축 | 시나리오 쌍 | 검토 내용 |
|---------|------------|----------|
| Mask 효과 | S1 vs S2 | PM10 mask 변수 추가의 영향 |
| 오염물질 추가 | S1 vs S3 | SO2, CO, O3, NO2 보조 정보 효과 |
| Mask 세분화 | S4 vs S5 vs S6 | 오염물질 포함 시 mask 전략 비교 (없음 / 전체 / 요약) |
| 기상변수 종류 | S7 vs S8 vs S9 | 기온·습도·기압 vs 강수 vs 둘 다 |
| 최대 특성 | S10 | 모든 후보 변수 통합 시 성능 |
